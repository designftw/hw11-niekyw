HW12 Writeup

Plan I intend to implement the severa, features for my chat app, including methods to customize the behavior when sending a message, and support for group chats.
  1. Copy to Clipboard: enable users to copy the current message to clipboard.
  2. Send to Self: send a copy of the current message as a direct message to themself.
  3. Scheduled Send: schedule a message to be sent later.
  4. Group Chat: enable users to invite others to a group for chat.
  5. Chat Tags: enable users to tag direct message or group chats
  6. Cosmetic and Animation: general animations to make the app look nice, including potentially darkmode.
Progress I have completed approximately half of these features.
  1. Copy to Clipboard: fully implemented. User can hover to the right of the send button for additional send options, including copy to clipboard.
  2. Send to Self: fully implemented. User can use the same dropdown to send private messages to themselves. For ease of accessing the thread to themselves, a "chat with self" button has also been added to private messaging when the recipient user is not the same as the logged in user.
  3. Scheduled Send: partially implemented. Code skeleton that changes button text (to inform state) at a given date time selection works. UI has also been implemented. Function integration with chat app to come.
  4. Group Chat: not started.
  5. Chat Tags: not started.
  6. Cosmetic and Animation: partially implemented. General animations are added as appropriate.
